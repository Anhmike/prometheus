h2. INTRODUCTION:

"sqlite2pgsql":http://github.com/reubano/sqlite2pgsql is a command line program that converts CSV files to GPX files for importing into GPSPhotoLinker or similar geotagging programs. "sqlite2pgsql":http://github.com/reubano/sqlite2pgsql is distributed under the terms of the GNU General Public License (GPL) and has been tested on the following configuration:
* MacOS X 10.5.8
* PHP 5.3.0
* Console_CommandLine 1.1.1
* GPSPhotoLinker 1.6.4

h2. REQUIREMENTS:

"sqlite2pgsql":http://github.com/reubano/sqlite2pgsql requires the following programs in order to run properly:

h2. INSTALLATION:

_*Create symlink of sqlite2pgsql in your ~/bin directory*_

<pre>if [-d $~/bin]; then mkdir ~/bin; fi
ln -s ./sqlite2pgsql/sqlite2pgsql.php ~/bin/sqlite2pgsql</pre>

_*Make sqlite2pgsql executable and add ~/bin to your PATH variable so that you can run 'sqlite2pgsql' from the command line*_

<pre>chmod 755 ~/bin/sqlite2pgsql
echo 'export PATH=~/bin:$PATH' >> ~/.profile
. ~/.profile</pre>

h2. USING sqlite2pgsql:

h3. Usage:

<pre>sqlite2pgsql [options] <csv_file></pre>

h3. Options:

<pre>  -d, --debug                 enables debug mode, displays the options and
                              arguments passed to the parser
  -o /path, --out-file=/path  location and name of resulting output file
                              (use '-' to write to stdout), defaults to the
                              project 'gpx_files' folder
  -r, --variables             enables variable mode, displays the value of
                              all program variables
  -v, --verbose               verbose output
  -h, --help                  show this help message and exit
  --version                   show the program version and exit

h3. Arguments:

@csv_file /path@

h3. Usage examples:

_*normal usage*_

@sqlite2pgsql ~/Downloads/gps_points.csv@

_*direct gpx file output to the user's downloads folder*_

@sqlite2pgsql -o ~/Downloads/gpx_file.gpx ~/Downloads/gps_points.csv@

h3. Where does sqlite2pgsql save files?

sqlite2pgsql saves resultant gpx files in the "gpx_files" sub-folder of the installed "sqlite2pgsql" folder.
